## Getting Started

#### Clone the repo

```
git clone https://github.com/hgq287/hg-react-native.git
cd hg-react-native
```

#### Install dependencies

```
npm install
```

#### Set up environment

```
cp .env.example .env
```

then, modify .env as needed (API base URL, environment keys, etc.)

#### Run the app

```
npm run ios      # for iOS
npm run android  # for Android
```
